# Proyecto-SQL---Base-de-Datos-Cooperativa.

(ITM - IV Semestre -Diseño de Base de Datos -Taller # 2  - Profesora Diana Rico)
TECNOLOGIA EN DISEÑO Y PROGRAMACIÓN DE SOLUCIONES DE SOFTWARE VIRTUAL

Desarrollo de consultas con una base de datos y tablas, ejemplo Cooperativa 


## Autor : Javier Orlando Delgado Piedrahita

Este repositorio contiene el desarrollo de 9 ejercicios prácticos aplicando consultas SQL en SQL Server.

##  Objetivo

Aplicar consultas avanzadas utilizando:

- INNER JOIN
- WHERE
- LIKE
- BETWEEN
- FORMAT
- IIF
- CASE
- LEN
- TRIM / LTRIM
- LEFT / RIGHT
- SUBSTRING

Siguiendo la metodología técnica establecida en clase, materiales de apoyo y videos de la profesora:
- Uso de alias
- Cualificación de campos
- Comentarios descriptivos
- Buenas prácticas de presentación

## Estructura del Proyecto

- '/database' → Script completo de creación de la base de datos.
- '/ejercicios' → Consultas organizadas por tema.
- Documento guía donde se realiza e taller.
- Video en la descrición.
- Trabajo hecho en SQL Server (versión actualizada)
